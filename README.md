# 📚 Book Collection Manager

An Android application built using **Kotlin** and **SQLite** to manage a personal book collection.
Users can add book details (title and author), store them locally in a database, and view all saved books in a structured list.

---

## 🚀 Features

* Add book title and author details
* Save book information using SQLite database
* Retrieve and display all stored books
* Clean and simple user interface
* Local data storage (offline support)

---

## 🛠 Tech Stack

* **Kotlin**
* **Android Studio**
* **SQLite**
* **RecyclerView / ListView**
* **XML Layouts**

---

## 📱 Screens

### Main Screen

* Enter Book Title
* Enter Author Name
* Save Book
* View All Books

### Book List Screen

* Displays all saved books from the database
